Como usar
Diagrama de nós

Mapa de conexões interativo

  • ⊕
    Use a roda do mouse para dar zoom e arraste o fundo para navegar
  • ○
    Cada círculo é um nó — cor indica a categoria (local, personagem ou evento)
  • ↗
    Linhas entre nós representam conexões históricas documentadas
  • ✦
    Clique em um nó para ver descrição, imagem e lista de conexões no painel direito
  • ⊙
    No painel, clique em qualquer conexão para saltar para aquele nó
  • ⠿
    Arraste nós individualmente para reorganizar o layout
Matrix de adjacência

Mapa completo de todos os pares

  • ▦
    Cada ponto dourado na matrix representa uma conexão entre dois nós
  • ⊕
    Use os botões + / − ou Ctrl + roda do mouse para dar zoom
  • ✦
    Clique em qualquer linha ou célula para destacar as conexões daquele nó
  • ⌖
    Passe o mouse sobre a matrix para ver os nomes do par linha × coluna
  • 🔍
    Use a barra de busca para encontrar um nó pelo nome e centralizar a view
  • ↗
    No painel direito, clique em qualquer conexão para navegar até ela
